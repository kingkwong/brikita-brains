Documentation for BrikitaBrains

Description

BrikitaBrains is all about creating games that test various skills that the brain uses. For our first game, The Stroop Test, we aim to test the effect of dissociating observations with the semantics we give the word. Words will be displayed along with colored "barrels." The goal is to say the color of the barrel, regardless of what the word says. Foundations are in place for the addition of future games.

Our website starts off with an option to either log in or sign up to use our service. Account information is stored in a centralized CSV file. Once the user signs in, they are presented with a variety of options, such as to play one of our games (for demostration purposes, there is only one game implemented at this time), see their own personal results, or see how they stack up against other users. All the games that are / will be implemented are simple, designed with SVG (scalable vector graphics). Future games will test various brain skills. For example, a test that ask the user to quickly click on a rectangle as soon as it changes color, or to distinguish an image of a sad person from images of happy people ( a test for emotional intelligence). The results will than be stored in a record in a csv file. Python will store and retrieve the info in the csv file and analyze the data.

Bugs

The security can be bypassed by typing your username in the query string

Someone spams the games by rapidly clicking enter. They get a very low accuracy, but also a very fast time. This time goes down in the leader boards

An animation glitch. After a word hovers through the screen it the very bottom of it can be seen in the top left corner

The program once showed the same results multiple times in user results. This glitch might occur again

Using commas in the username will cause the system to crash since it will confuse the csv database containing the username and their scores

YOU NEED TO PLAY AT LEAST 2 GAMES FOR THE POLYLINE GRAPH IN THE RESULTS PAGE TO WORK. Here is an example of someone else's page with polyline graph's: Nikita's user results

If you get 0 correct on any of the three sections of the stroop test, the global leaderboards and the user results for that account will be broken... Thanks to Kenny Lin (p.10) for finding this!!!
